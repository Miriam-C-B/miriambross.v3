Front End Development

Wordpress sites are easy to set up, maintain and use. But I had big plans for my author business from the start. I needed plug-ins to embed newsletter sign-ups and direct payments, CSS to customize my colour schemes and how to do you make custom themes anyway? So I learned how to code. I tried free courses on Udemy first and even though they were alright, I wanted to learn in a more structured way.

Luckily, I stumbled over Skillcrush and their Front End Development course. They made coding look like fun. The best part: the company was founded by a woman and most of the employees are female. This helped enormously with the biggest obstacle I faced, the utterly wrong belief that coding was not for me. Of course it is.

I've built several websites since, every one of them a little bit better than the last. Some are no longer online (cause they were not that great), others are still in my Repo Gallery on GitHub.

I am shelving this project for now. JS React will have to wait for a future scanner cycle. But I haven't stopped coding altogether! I'm just using a more playful approach with an app called Mimo. It's cute, it's fun and it only takes me 15 minutes a day. I love it!

I practiced:

  • looking up everything I don't know
  • shipping
  • asking for help when I was stuck
  • attention to detail
  • taking regular breaks
  • choosing the easy option, instead of the unnecessarily difficult one

I learned:

  • HTML5
  • CSS3
  • Git and GitHub
  • JavaScript
  • Bootstrap 5
  • technology is for everyone.
  • a project doesn't have to be perfect to be shared! You can always improve it later.
