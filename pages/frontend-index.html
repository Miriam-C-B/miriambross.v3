Front End Development

Wordpress sites are easy to set up, maintain and use. But I have big plans with my author business. I need plug-ins to embed newsletter sign-ups and direct payments down the line. So I learned how to code. I tried free versions on Udemy first and even though the courses war alright, I wanted to learn in a more structured way.

Luckily, I stumbled over Skillcrush and their Front End Development course. They made coding look like fun. The best part for me: the company was founded by a woman and most of the employees are female. It helped overcome the biggest obstacle - the utterly wrong belief that coding was not for women. Of course it is and always has been!

I've written several websites since I started the course. Every one of them a little bit better than the other. Some are no longer online (cause they were not that great), others are still on GitHub. This particular site is like most things in life a work in progress.

I practiced:

  • looking up things I don't know
  • asking for help when I got stuck
  • attention to detail
  • taking regular breaks
  • shipping

I learned:

  • HTML5
  • CSS3
  • Git and GitHub
  • JavaScript
  • Bootstrap 5
  • technology is for everyone.
  • that a project doesn't have to be perfect to be shared! You can always improve it later on.
